# DocuMaster Pro

A multi-tenant Document Management System built with Node.js, React, and MongoDB.

## Features
- Multi-company support (header-based database switching)
- JWT authentication with role-based permissions
- Document upload/download with versioning
- Contract management with e-signature support
- Template library for documents and contracts
- Folder-based organization
- Comprehensive audit logging
- Modern React frontend with dark theme

## Tech Stack
- **Backend:** Node.js + Express + MongoDB
- **Frontend:** React + Vite + Tailwind CSS
- **Database:** MongoDB (multi-tenant)
- **AI Model:** Qwen Coder Next 80B (local on DominusPrime)

## Getting Started

### Backend
\`\`\`bash
cd backend
npm install
npm start
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
